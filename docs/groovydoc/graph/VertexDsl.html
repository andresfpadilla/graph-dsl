        • Vertex vertex(Map<String, ?> map)

          Creates or updates a Vertex in this graph. The map may contain configuration for the vertex. Default configuration can be:

          id
          id of the vertex to create or update
          changeId
          new id for the vertex
          connectsTo
          list of vertex keys the vertex should connect to. Edges will be created with edge.one equal to the vertex id and edge.two equals to the 'connectTo' id.
          Additional entries may be added by type of the graph. The map must contain id for this method.
          Parameters:
          map - configuration of Vertex
          Returns:
          the resulting Vertex

        • Vertex vertex(Object id, @DelegatesTo(VertexSpec) Closure closure)

          Creates or updates a Vertex in this graph with the given id. closure is used to further customize the vertex and graph. In the closure getting and setting properties delegates to the vertex. Methods also delegate to the vertex.

          Variables accessible within the closure:

          graph
          Graph vertex was added to
          vertex
          Vertex added to graph

          By default there are several methods added in the closure.

          void changeId(Object id)
          Changes id of vertex
          void changeId(NameSpec newName)
          renames the vertex using a NameSpec
          void connectsTo(Object... keys)
          Connects the vertex to other vertices. If they do not exist they are created.
          void connectsTo(ConfigSpec... specs)
          Connects the vertex to other vertices. If they do not exist they are created. This method allows for arbitrarily nested configurations.

          Types may add variables and methods to the passed in closure.

          Parameters:
          id - the id of the Vertex to find or create.
          closure - configuration for graph and vertex
          Returns:
          The resulting Vertex.
