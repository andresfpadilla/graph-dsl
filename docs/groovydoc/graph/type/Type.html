Package: graph.type

[Groovy] Interface Type

  • Classes implementing this interface are able to change the behavior of a Graph. A Type can change the base class of all Vertex and Edge objects to implement the behavior needed but a Type should not add traits to delegates. A Type is required to provide factories for Edge, Vertex, EdgeSpec, and VertexSpec. Subclasses can be returned by these factory methods which allow different behavior in the graph. For instance Edge methods can return DirectedEdge objects and UndirectedVertexSpec methods can return DirectedVertexSpec objects.
