Package: graph.type

[Groovy] Class AbstractVertexSpec

      • graph.type.AbstractVertexSpec

      • Base implementation of a VertexSpec. Type packages can inherit this class to implement default methods in VertexSpec. Add new dsl properties for Vertex dsl to dslProperties. This will prevent them from being added as entries to the Vertex.
