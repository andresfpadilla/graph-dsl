[Groovy] Class VertexSpecCodeRunner

    • graph.type.undirected.VertexSpecCodeRunner

    • Delegate of the runnerCode closure in GraphVertexSpec. This provides methods and properties that can be used in the closure. Method and property missing is delegated to the Vertex.
