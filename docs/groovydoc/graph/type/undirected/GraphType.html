    • All Implemented Interfaces and Traits:
      Type

      class GraphType
extends Object

      Implements an undirected graph. There can only be one Edge between any two vertices. When traversing a graph an Edge is adjacent to a Vertex if it's one or two property equals the name of the Vertex.
