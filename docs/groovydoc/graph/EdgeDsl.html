        • Edge edge(Object one, Object two)

          Creates or finds an Edge between two Vertex objects returning the Edge. The Vertex objects are identified by the params one and two. If the Vertex objects do not exist they will be created.

          throws:
          IllegalArgumentException when one or two are invalid
          Returns:
          the resulting Edge.
          Parameters:
          one - the key of the first Vertex.
          two - the key of the second Vertex.

        • Edge edge(Map<String, ?> map)

          Creates or finds an Edge between two Vertex objects returning the Edge.The map may contain configuration for the edge. Default configuration options are:

          one
          key of the first Vertex
          two
          key of the second Vertex
          changeOne
          If the edge already exists edge.one will become changeOne. Otherwise edge.one is set to changeOne instead of one when it is created.
          changeTwo
          If the edge already exists edge.two will become changeTwo. Otherwise edge.two is set to changeTwo instead of two when it is created.
          Additional entries may be added by the type of the graph.
          throws:
          IllegalArgumentException when one or two are invalid
          Returns:
          the resulting Edge.
          Parameters:
          map - used to create an Edge.
          See Also:
          edge(String one, String two, Closure closure)

        • Edge edge(Object one, Object two, Closure closure)

          Creates or finds an Edge between two Vertex objects returning the Edge. Closure is used to further customize the edge and graph. In the closure getting and setting properties delegates to the edge. Methods also delegate to the edge.

          Variables accessible within the closure:

          graph
          Graph edge was added to
          edge
          Edge added to graph
          Attempting to set one or two will throw an exception. Use the changeOne and changeTwo methods.

          By default there are several methods available in the closure.

          void changeOne(String changeOne)
          Changes edge.one
          void changeOne(NameSpec changeOne)
          Changes edge.one
          void changeTwo(String changeTwo)
          Changes edge.two
          void changeTwo(NameSpec changeTwo)
          Changes edge.two
          void traits(Class... traits)
          Applies a trait the edge's delegate
          Plugins may add variables and methods to the passed in closure.
          throws:
          IllegalArgumentException when one or two are invalid
          Returns:
          the resulting Edge.
          Parameters:
          one - the key of the first Vertex.
          two - the key of the second Vertex.
          closure - to run.
