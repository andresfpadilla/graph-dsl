Package: graph

[Groovy] Class VertexSpecCodeRunner

    • graph.VertexSpecCodeRunner

    • Delegate of the runnerCode closure in VertexSpec. This provides methods and properties that can be used in the closure. Method and property missing is delegated to the Vertex.

          This object becomes the delegate of closure and then closure is called. The methods in this object become available in the closure along with the properties vertex and graph.
