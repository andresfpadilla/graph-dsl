Package: graph

[Groovy] Class Graph

    • graph.Graph

    • An implementation of a Graph. A Vertex is identified in this Graph using the vertex name property. A Edge is identified by the names of the to vertices it connects.

      Vertex and Edge objects are added using the vertex and edge methods. These methods will always create objects if not present to reduce the code needed to express a graph and to ensure the integrity of the graph. There are a few styles that may be used to express vertices and edges in this Graph. See the edge and vertex methods for more details.

      The default behavior is that of an undirected graph. There can only be one Edge between any two vertices. When traversing a graph an Edge is adjacent to a Vertex if it's one or two property equals the name of the Vertex.

      Plugins may be applied to this graph to change its behavior and the behavior of the vertices and edges. For more information on plugins see Plugin.

        • void deleteEdge(String one, String two)

          Removes an Edge from edges where an edge created by the edgeFactory equals an edge in edges. Using the edgeFactory ensures the edge removed matches the definition of an edge for this graph. If a plugin changes the definition of an edge, for example to DirectedEdge, this method will still work as expected. It will remove the edge where edge.one == one and edge.two == two. Keep in mind, in the case of the base Edge object edge.one can also equal two and edge.two can also equal one.

          Parameters:
          one - name of first vertex
