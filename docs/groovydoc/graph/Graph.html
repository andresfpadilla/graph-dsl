Package: graph

[Groovy] Class Graph

    • graph.Graph
    • All Implemented Interfaces and Traits:
      EdgeDsl, TraversalDsl, VertexDsl

      class Graph
extends Object

      An implementation of a Graph. A Vertex is identified in this Graph using the vertex name property. A Edge is identified by the names of the to vertices it connects.

      Vertex and Edge objects are added using the vertex and edge methods. These methods will always create objects if not present to reduce the code needed to express a graph and to ensure the integrity of the graph. There are a few styles that may be used to express vertices and edges in this Graph. See the edge and vertex methods for more details.

      All graphs have a Type. All Vertex, Edge, VertexSpec, and EdgeSpec objects must be created by the Type. This is to ensure the Graph always has the behavior specified by the Type.

      The default behavior is that of an undirected graph. This is implemented by GraphType.

        • void deleteEdge(Object one, Object two)

          Removes an Edge from edges If a type changes the definition of an edge, for example to DirectedEdge, this method will still work as expected. It will remove the edge where edge.one == one and edge.two == two. Keep in mind, in the case of the base Edge object edge.one can also equal two and edge.two can also equal one.

          Parameters:
          one - key of first vertex
          two - key of second vertex
