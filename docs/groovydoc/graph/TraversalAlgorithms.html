          Uses the specification spec to performs a pre-order depth first traversal on a component of graph returning the results. The traversals starts at spec.root. action must return a TraversalState. As the traversal runs, action is called with each vertex as a parameter. If action returns TraversalState.STOP the algorithm stops.

          spec is the specification for the algorithm. spec is returned so it can be reused on other components of graph. spec must contain the following.

          Object root
          root of traversal. root will change for each recursive call in this algorithm. (not null)
          Map<Object, TraversalColor> colors
          colors for each vertex. This is updated as the traversal runs. If the color of a vertex is not found it is assumed to be TraversalColor.WHITE. (not null, can be empty)

          The following will be added to spec by this algorithm.

          TraversalState state
          state of traversal returned by action.
          Parameters:
          graph - to perform algorithm on
          spec - specification of algorithm
          action - taken on each vertex
          Returns:
          resulting specification. will always return the spec param.
