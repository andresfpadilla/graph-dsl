        • Map traversal(Closure algorithm, Object root, Map<Object, TraversalColor> colors, Closure action)

          Performs a full traversal with the given algorithm on all components of this Graph. algorithm must accept the following parameters.

          algorithm must meet the following:

          • each call to algorithm must visit part of the graph by updating spec.colors. This will eventually cause calls to Graph.getUnvisitedVertexKey to return null which signals the end of the traversal.
          • must call action on each vertex or edge
          • must stop when action returns TraversalState.STOP
          • must throw exception when action does not return a TraversalState

          Returns:
          results containing current root, roots of all components, colors, and results added by algorithm
          Parameters:
          algorithm - type of traversal to run on each component
          root - root of component to start traversal
          colors - starting set of colors. Will not be modified.
          action - action to perform by algorithm (on each edge or vertex)
